# Data Structures

## First In First Out

Used to link producers to consumers

- Producer: fifo_put()
- Consumer: fifo_get()

### Examples of Producers and Consumers

Producer:
- Keyboard input
- Program sends message

Consumer:
- Program that displays it
- Program receives message
